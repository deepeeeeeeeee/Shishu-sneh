Baby Care & Health Tracker is an Android application developed using Kotlin and Jetpack Compose that helps parents manage baby healthcare activities such as growth tracking, medicine reminders, and medical record storage.
Features
📊 Growth Analytics with charts
💊 Medicine Reminder System
🩺 Digital Health Records
⚡ Quick Actions Dashboard
🔔 Notifications Support
⚙️ User Settings & Preferences
Tech Stack
Kotlin
Jetpack Compose
Android Studio
Material Design
MVVM Architecture
Objective

The main objective of this project is to provide parents with a simple and organized mobile solution for monitoring baby health, tracking growth progress, scheduling medicines, and storing important medical records.
